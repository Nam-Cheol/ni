# Interaction contract

## Interaction mode

Human to system.

## Product interaction

The current interaction is a planning conversation. The model may draft planning
records and focused questions, but it may not implement the dashboard or declare
readiness from judgment alone.

## User control

The user resolves blocker questions. The CLI decides readiness. `ni end` is the
only path to lock, and `ni run` compiles a prompt only after a valid lock.
