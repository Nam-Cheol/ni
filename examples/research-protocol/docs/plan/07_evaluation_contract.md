# Evaluation contract

## EVAL-001: Sampling reproducibility

Method: an independent reviewer uses the protocol to select eligible street segments from a supplied map and records mismatches.

## EVAL-002: Ethics boundary review

Method: a reviewer classifies each planned data item as public environmental observation, participant contact, or personally identifying data.

## EVAL-003: Analysis reproducibility review

Method: a reviewer confirms variables, comparison method, exclusions, and evidence package are enough to rerun the analysis.
