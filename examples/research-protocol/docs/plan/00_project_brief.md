# Project brief

## Product type

research_protocol

## Delivery surfaces

- document

## Purpose

Neighborhood Cooling Study Protocol is a documented research method for comparing street-level cooling interventions before any field execution begins.

## Problem

Research planning often mixes hypothesis, sampling, ethics, and analysis choices. This example proves ni can lock those choices as a product planning contract without assuming software, web UI, or runtime execution.

## Success definition

The plan is successful when `ni status` reports READY, `ni end` writes `.ni/plan.lock.json`, and `ni run --target human-team` produces a bounded handoff prompt for a research team.
